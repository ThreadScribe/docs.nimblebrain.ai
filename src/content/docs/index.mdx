[ThreadScribe.ai](https://www.threadscribe.ai) is a powerful AI copilot for customer oriented teams.

ThreadScribe integrates seamlessly with popular tools such as Slack and Notion, providing a centralized, powerful search engine to enhance your team's efficiency and productivity.

## Features

ThreadScribe offers powerful features via chat based interfaces:

<CardGrid>
  <Card title="Summarize" icon="document">
    Instantly generate summaries to ensure key actions and decisions are never lost.
  </Card>
  <Card title="Search" icon="magnifier">
    Easily search across multiple data sources and rapidly find answers.
  </Card>
  <Card title="Automations" icon="magnifier">
    Stay up to date with automated thread and channel digests and workflows.
  </Card>
  <Card title="Slack-based" icon="slack">
    Keeps your agents focused and work right in Slack, limiting context switching.
  </Card>
</CardGrid>
